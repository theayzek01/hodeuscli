## 🇺🇸 English Version

Hodeuscli is the world's most advanced, luxury-grade terminal coding assistant, designed for performance and aesthetic excellence. Powered by **Hodeus Studio**, it features a high-fidelity **Antigravity TUI** with millisecond-precision timers, real-time status spinners, and a sophisticated bilingual interface.

### 📦 Packages

| Package | Description |
|---------|-------------|
| **[@games-coder/hodeuscli-ai](packages/ai)** | Unified multi-provider LLM API (OpenAI, Anthropic, Google, etc.) |
| **[@games-coder/hodeuscli-agent-core](packages/agent)** | Agent runtime with tool calling and state management |
| **[@games-coder/hodeuscli-coding-agent](packages/coding-agent)** | Flagship Hodeuscli Interactive Terminal Experience |
| **[@games-coder/hodeuscli-tui](packages/tui)** | Premium Terminal UI library with high-perf differential rendering |
| **[@games-coder/hodeuscli-web-ui](packages/web-ui)** | Web components for modern AI chat interfaces |
| **[@games-coder/hodeuscli-pods](packages/pods)** | CLI for managing vLLM deployments on GPU pods |

### 🛠️ Quick Start

```bash
npm install          # Install all dependencies
npm run build        # Build all packages
npm run check        # Lint, format, and type check
./hodeuscli-test.sh  # Run Hodeuscli from source
```

## 🇹🇷 Türkçe Versiyon

Hodeuscli, performans ve estetik mükemmellik için tasarlanmış, dünyanın en gelişmiş, lüks sınıf terminal kodlama asistanıdır. **Hodeus Studio** tarafından desteklenen bu sistem, milisaniye hassasiyetinde zamanlayıcılar, gerçek zamanlı durum göstergeleri ve sofistike bir kullanıcı arayüzü sunan yüksek sadakatli **Antigravity TUI** deneyimine sahiptir.
