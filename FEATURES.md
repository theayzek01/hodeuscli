# HODEUSCLI Features & Capabilities

Hodeuscli (Antigravity Edition) is more than just a terminal assistant; it's a comprehensive AI-powered development environment designed for modern engineers.

## 🚀 Core Capabilities

### 🌐 Live Web Search

Stay up-to-date without leaving your terminal. Hodeuscli integrates with leading search engines to provide:
- **Real-time Documentation**: Search for the latest API changes or library usage.
- **Current News**: Get updates on technology and world events.
- **Live Data**: Fetch current statistics, weather, or market data.
- **How to use**: Simply ask a question that requires external knowledge (e.g., "What is the latest version of React?").

### 📁 Local File Access (The "Coding Partner")

Hodeuscli acts as a true pair programmer with direct access to your local filesystem (with your permission):
- **Full Context**: Reads multiple project files to understand architecture and dependencies.
- **Code Generation**: Writes full source files or applies precise diffs.
- **Bug Fixing**: Analyzes error logs and source code to identify and resolve issues.
- **Project Indexing**: Automatically builds a local index of your code for faster retrieval.

### 🧠 Personal Knowledge Base (RAG - Memory)

Store and retrieve your own private documentation:
- **Multi-format Support**: Upload and index PDF, DOCX, and TXT files.
- **Semantic Search**: Find relevant information across your entire library based on meaning, not just keywords.
- **Long-term Memory**: The agent remembers your project-specific conventions and past interactions.

### ⚡ Planning vs. Fast Modes

Optimize your workflow based on the task complexity:
- **Planning Mode**: High-reasoning state where the agent researches, plans, and requests feedback before executing large changes.
- **Fast Mode**: Instant execution for simple tasks, questions, or small code tweaks.
- **Toggle**: Use `Shift+Tab` to cycle between modes instantly.

## 🎨 Advanced TUI Features

### 📋 Smart Paste Management

Handling large code blocks in a terminal is now seamless and beautiful:
- **Language Detection**: Automatically identifies `Python`, `HTML`, `JavaScript`, `TypeScript`, `CSS`, and `JSON`.
- **Visual Styling**: Displays distinct colored icons and language labels for pasted markers.
- **Efficiency**: Keeps the chat history clean by using markers like `[py] [paste #1 +123 lines]` instead of dumping thousands of lines.

### ⌨️ Terminal Integration & Shortcuts

- **Direct Bash**: Run commands using `!` (with context) or `!!` (isolated).
- **Autocomplete**: Intelligent fuzzy search for files and slash commands.
- **Model Cycling**: Quickly switch between different LLMs for different tasks.
- **Image Support**: Paste images from the clipboard directly into the terminal for analysis.

## 🇹🇷 Özellikler ve Yetenekler (Türkçe)

### 🌐 Canlı Web Tarama

Terminalinizden ayrılmadan güncel kalın:
- **Canlı Dokümantasyon**: En yeni API değişikliklerini veya kütüphane kullanımlarını arayın.
- **Güncel Haberler**: Teknoloji ve dünya olayları hakkında güncellemeler alın.

### 📁 Yerel Dosya Erişimi ("Kod Partneri" Modu)

Hodeuscli, yerel dosya sisteminize doğrudan erişimi olan gerçek bir eş programcı gibi çalışır:
- **Tam Bağlam**: Proje dosyalarınızı okuyarak mimariyi ve bağımlılıkları anlar.
- **Kod Yazma ve Hata Düzeltme**: Hata günlüklerini ve kaynak kodunu analiz ederek sorunları giderir.

### 🧠 Kişisel Bilgi Bankası (RAG)
Kendi özel dokümanlarınızı yükleyin ve asistanın bunları hatırlamasını sağlayın:
- **Çoklu Format**: PDF, DOCX ve metin dosyalarını indeksleyin.
- **Anlamsal Arama**: Anahtar kelimelere takılmadan anlam bazlı arama yapın.

### ⚡ Planlama ve Hızlı Mod
İş yüküne göre iş akışınızı optimize edin:
- **Planlama Modu**: Büyük değişiklikler öncesinde araştırma yapan ve onay isteyen yüksek muhakeme durumu.
- **Hızlı Mod**: Basit görevler ve sorular için anında yanıt.
- **Geçiş**: Modlar arasında geçiş yapmak için `Shift+Tab` tuş kombinasyonunu kullanın.
